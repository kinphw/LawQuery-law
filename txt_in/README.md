# txt_in — 법령 본문 txt 입력 폴더

국가법령정보센터(law.go.kr)에서 **단별 본문을 복사해 txt로 저장**하는 곳.
파서가 여기 파일들을 읽어 조문 분리 → ID 부여 → rdb 연계까지 자동 생성한다.

## 파일명 규약 (있는 단만)
| 파일 | 단 | tier |
|---|---|---|
| `a.txt` | 법 | a |
| `e.txt` | 시행령 | e |
| `s.txt` | 감독규정 | s |
| `r.txt` | 시행세칙 | r |

- 새 법마다 하위 폴더로 묶어도 됨: `txt_in/<코드>/a.txt …` (예: `txt_in/g/a.txt`)
- 각 txt 첫머리에 법령정보센터의 **법령명·[시행…]·[법률 제…호…]** 헤더가 있으면 meta로 파싱됨
- 실제 본문 txt는 git 추적 제외(`txt_in/**/*.txt`). 이 README만 추적.
